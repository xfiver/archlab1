## Задание 1. Вариант 21
Модель взаимодействия открытых систем (OSI).

OSI (Open system interconnections) – это  концептуальная модель, описывающая то, как передаются данные между устройствами по сети.

Имеет 7 уровней:
* Прикладной (между сетью и прикладными приложениями, например браузер, почта). Отвечает за доступ к сетевым службам.
* Представительский уровень (кодирование/декодирование/шифрование/сжатие данных). Отвечает за перевод передаваемых данных – например ASCII в  UNICODE
* Сеансовый уровень (создание, поддержание, завершение сессии). Отвечает за диалог между устройствами.
* Транспортный. Отвечает за передачу данных, зависит от протокола (TCP или UDP)
* Сетевой. Отвечает за определение маршрута (IP адреса, роутинг)
* Канальный. Отвечает за обработку ограничений и ошибок, например как долго идет "общение" между двумя машинами.
* Физический. Отвечает за передачу потока необработанных битов. (Провода.)

Принципы работы: инкапсуляция и декапсуляция.
При отправке данные проходят уровни с 7 (прикладной) по 1 (физический), на каждом уровне собирая заголовки. При получении данных, устройство проводит их в обратном порядке, снимая заголовки.


## Задание 2. Вариант 16.
В чем заключаются отличия между SRAM (Static Random Access Memory) и DRAM (Dynamic Random Access Memory)?

Коротко:
SRAM -- дорогая, но очень быстрая память, обычно использующаяся как кеш память процессора (L3, L2, L1).
DRAM -- более дешевая и менее быстрая память, обычно использующаяся как оперативная память.

Длинно:

SRAM (static random access memory) -- схема на 4-6 транзистора, которая статично хранит биты, пока подается питание. Пока находится в статичном режиме, потребляет мизерное количество энергии, зато при переключении потребление сильно поднимается. Ввиду того, что в одной ячейке 6 транзисторов имеет намного большую стоимость.

DRAM (dynamic random access memory) -- схема на 1 транзистор и 1 конденсатор. Хранит данные постоянным заряжением конденсатора. Ввиду этого всё оставльное.




## Задание 3. Вариант 1.

Определите тип ЭВМ по предложенному описанию:
Вычислительная машина непрерывного действия, работает с информацией, представленной в непрерывной форме, то есть в виде непрерывного ряда значений какой-либо физической величины (чаще всего электрического напряжения).

Это аналоговая вычеслительная машина, потому что она непрерывно действует и обрабатывает электрическое напряжение как основной источник данных.
